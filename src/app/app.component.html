you probably want to do this on a computer so that you can copy all messages from molbot in one go, its a bit of a pain otherwise, but possible
The data from molbot is not really consistent, so it is likely some people will have issues. The issues will manifest themselves as nothing will happen when you hit the add button, it it works the box will empty and the last charecter on the list will be printed below the buttons (to help finding the next message if you do this on a cell..) if you encounter any issues, or have questions, reach out to dan... and, good luck!
  • Update you roster in molbot
  • generate the roster data from your molbot channel with !charlist -c 131 -m level
  • copy all messages (or one, or a few) into the text area below and click the add button
  • when you have added all the messages, hit the download button and you can save the cvs file locally
  • logon to msf.gg and under your profile and roster, import the csv file you saved
  • Now the data from molbot is added, but it was most likely incomplete so you need to manually add the missing levels etc..
  • good luck and have fun :-)
